## Design Direction

- Fun but not too informal: friendly learning energy, tiny doodle accents, and warm colors without turning the app into a toy.
- Productive learning workspace: clear navigation, readable content, progress visibility, and stable AI generation states.
- Visual palette: off-white canvas, charcoal text, blue primary actions, green progress and completion states, amber/coral accents.
- Core UI patterns: compact app shell, guide cards, ordered topic rows, markdown-style lesson reading, bottom mobile tabs, and clear account/auth states.
- Card corners stay modest, with clean spacing and soft shadows.
