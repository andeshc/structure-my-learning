Self-Attention

Transformer Architecture

Overview

Self-attention is the mechanism that lets every token compare itself with every other token in the same sequence. Instead of reading a sentence strictly left to right, the model builds a context-aware view of each token.

Core idea

For each token, the model creates query, key, and value vectors. The query asks what this token needs, keys describe what other tokens offer, and values carry the information that may be blended into the final representation.

Analogy

Imagine a study group where each student asks a question, listens to the most relevant classmates, and writes a better answer using what they heard.

Diagram to Imagine

Picture tokens in a row with thin lines connecting every token to every other token. Thicker lines represent stronger attention weights.

Brief summary

Self-attention turns isolated token embeddings into contextual representations by learning which relationships matter inside a sequence.
